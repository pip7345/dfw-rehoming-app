# Core Requirements

## 1. Rehomer Accounts and Authentication
- Rehomers can register and login using Facebook OAuth or username/password.
- Each rehomer has a profile with contact preferences and payment status.

## 2. Paid Listing Access
- Rehomers pay for access to create and manage listing pages.
- Payment grants access to list litters or individual dogs for a defined period.

## 3. Litter Listings
- Rehomers create a page for each litter with:
  - General details: breed, date of birth, expected availability date, location
  - Text description of litter characteristics
  - General photos of the litter
- Individual puppies within a litter can be advertised separately with:
  - Name or identifier
  - Gender
  - Markings or distinguishing features
  - Availability status: available, reserved, sold, or custom date

## 4. Single Dog Listings
- Rehomers can also list individual dogs (not part of a litter) that need rehoming.
- Same detail fields as litter entries: breed, age, description, photos, availability.

## 5. Contact Method Preferences
- Rehomers specify how they want to be contacted:
  - Facebook Messenger (preferred option)
  - Email
  - Phone
  - Contact form
- Public viewers see the rehomer's preferred contact method on each listing.

## 6. Availability Management
- Rehomers can mark dogs or puppies as:
  - Available now
  - Available on a specific future date
  - Reserved or pending
  - No longer available or adopted
- Status updates are reflected immediately on public listings.

## 7. Photo Uploads
- Rehomers upload photos for litters and individual dogs.
- Support multiple images per listing with ordering and captions.
